# C Array Algorithm Lab

A structured repository containing implementations of fundamental array and matrix algorithms in the C programming language. The programs demonstrate essential computational techniques including array traversal, statistical analysis, searching, sorting, and multidimensional matrix operations.

---

## Repository Objectives

- Understand one-dimensional and two-dimensional arrays.
- Implement common searching and sorting algorithms.
- Perform statistical computations on array elements.
- Explore matrix manipulation techniques.
- Build a strong foundation for Data Structures and Algorithms using C.
